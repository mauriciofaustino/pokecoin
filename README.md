# Pokecoin

Uma aplicação para acompanhar o rendimentos dos seus NFTs Pokemons 

Tech stack: Spring-boot, SpringMVC, SpringSecurity, Thymeleaf, PostgreSQL

Para rodar:
```bash
./mvnw spring-boot:run 
```
